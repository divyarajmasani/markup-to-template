Gutenberg to InnerBlocks Converter

This tool converts WordPress Gutenberg block markup (HTML with comments) into a template that can be used with InnerBlocks.

Simply paste your Gutenberg HTML in the left panel and click "Convert to InnerBlocks Template".

Example Usage:

      
        // In your block registration:
        registerBlockType('my-plugin/my-block', {
            // ... other settings
            edit: ({ attributes, setAttributes }) => {
                return (
                    <div className="my-custom-block">
                        <InnerBlocks
                            template={TEMPLATE}
                            templateLock="all"
                        />
                    </div>
                );
            },
            save: ({ attributes }) => {
                return (
                    <div className="my-custom-block">
                        <InnerBlocks.Content />
                    </div>
                );
            }
        });
